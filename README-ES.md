# LinAlert

# ¡AVISO!:
Este archivo Léame está en desarrollo. ¡Podría cambiar con el tiempo!

# Acerca de
LinAlert es, básicamente, una radio meteorológica personalizada para sistemas Debian.

# Características
* Puede crear las mismas cabeceras
* APIs de IEMBot y NWS para alertas
* ¡Compatibilidad con inglés! (Compatibal with English! Wait, if you understand English and aren't a native Spanish person, why are you here?)
